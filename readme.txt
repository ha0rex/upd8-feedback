A simple & lightweight, multilingual WordPress plugin to collect your users 5 star rating like feedback.

== Description ==

Are you interested in your user's Feedback? Would you like to know, if they like your product or not? Create an easy and straightforward Quiz like Feedback form, and plcae it anywhere on your website: on a page, on the sidebar, in a lightbox, or wherever you prefer by this shortcode:
[upd8-feedback id="FORM-ID"]

== Installation ==

How to install this plugin?

1. Upload the plugin files to the `/wp-content/plugins/upd8-feedback` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Feedback menu to set up your forst Form
4. Place shortcode: [upd8-feedback id="FORM-ID"]


== Frequently Asked Questions ==

= Where can I check the stats of my form fills? =

All the form fills and their averages are displayed on the bottom of form page.
